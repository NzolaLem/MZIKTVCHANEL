Add these exact Environment Variables in Vercel.

Project Settings -> Environment Variables -> Add Environment Variable

Do not create variables named "Value" or "Important".

ADMIN_PASSWORD=choose-a-private-password-for-the-admin-dashboard
TOKEN_SECRET=use-a-long-random-secret-text-here
SUPABASE_PROJECT_ID=lzepyzvisjkwnytxopmd
SUPABASE_URL=https://lzepyzvisjkwnytxopmd.supabase.co
SUPABASE_SERVICE_ROLE_KEY=paste-your-new-rotated-supabase-secret-key-here

Important:
- Add each line above as one separate Vercel variable.
- The text before "=" is the Vercel variable name.
- The text after "=" is the Vercel value.
- Do not add API_PORT in Vercel.
- Do not name the Supabase service key VITE_SUPABASE_SERVICE_ROLE_KEY.
- Because the old Supabase key was pasted in chat, rotate it in Supabase and paste the new one as SUPABASE_SERVICE_ROLE_KEY.
